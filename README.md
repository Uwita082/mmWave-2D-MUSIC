# mmWave-2D-MUSIC
Super-resolution radar point clouds from the TI IWR6843ISK using 2D MUSIC. Parses raw ADC captures from the DCA1000EVM and runs a full range–Doppler–angle pipeline with CFAR detection, TDM-MIMO Doppler-phase compensation, and azimuth-only or joint azimuth–elevation DOA estimation.
